# Ubuntu-Configure-Script
Scripts to configure a fresh install of Ubuntu 14.04.

### Settings:
* Adds myself to sudoers.d
* Adds my internal devbox to fstab

### Installs:

* Google Chrome
* Sublime Text 2
* Kubuntu Desktop
* OwnCloud Client
* Git
* CIFS-Utils
* Kubuntu Desktop
* Spotify
