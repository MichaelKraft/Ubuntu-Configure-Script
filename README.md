# Ubuntu-Configure-Script
Scripts to configure a fresh install of Ubuntu 14.04.

## configure-ubuntu-box.sh

### Installs:

* Google Chrome
* Sublime Text 2
* Kubuntu Desktop
* OwnCloud Client
* Git
* CIFS-Utils
* Kubuntu Desktop
* Spotify

*More will likely be added down the line.*

### Other

* Adds a link to my devbox
* Adds me to sudoers
